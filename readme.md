# 地震データベースWeb

## はじめに
本Webサイトは2023年03月03日以降の地震データを検索・表示するものです。地震データは私個人が趣味でGASを用いてスプレッドシートに収集し続けているものであり、場合によっては誤り等を含むことがあります。また本Webサイトを使用することにより生じた損害については一切の責任を負いません。

## 機能及び仕様
### 検索
本Webサイトではこれまで収集した地震情報について「期間」「地域（震央地名）」「マグニチュード」「震度」の条件により検索することができます。  

### 地図表示
地図上に表示されるピンマークをクリックまたはタップすることにより、その地震の詳細情報が閲覧できます。  

### デフォルト表示
アクセス時に表示されるデフォルト画面では過去30日間の地震データが地図上に表示されるとともに、詳細パネルに直近3件の地震情報が表示されます。  

### 仕様
データ収集の都合上、2026年04月25日以前の地震データについて緯度経度は代表地点を使用しているため不正確です。重なって表示されますが、拡大縮小をした際クラスター表示されたものをクリックすることで個別選択が可能です。  

地震規模（マグニチュード）が精査により更新された場合、同一日時の地震情報が複数表示されます。  

データ更新はおおよそ20分毎です。データ更新後はページ再読み込みにより変更が反映されます。

## その他
- 使用している地震情報の一次ソースは気象庁です。
- 収集した地震情報はGASスクリプトによって私個人のスプレッドシートに蓄積されます。
- 気象庁によるデータ更新及びGASスクリプトの実行タイミングにより地震情報の更新に時間がかかる場合があります。
