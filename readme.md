# 地震データベースWeb
2026年05月25日更新（v1.1.0準拠）

## はじめに
本Webサイトは2023年03月03日以降の地震データを検索・表示するものです。地震データは私個人が趣味でGASを用いてスプレッドシートに収集し続けているものであり、場合によっては誤り等を含むことがあります。また本Webサイトを使用することにより生じた損害については一切の責任を負いません。  
2026年04月24日以降の地震データについては緯度経度を収集・記録しているため正確な震央分布が表示されます。

## 目的
本Webサイトは過去の地震データを可視化し、最大震度や震央地名、マグニチュード等の任意の条件により絞り込み検索し確認することを容易に行える手段の提供を目的としています。  
またv1.1.0より震度観測点の表示に対応しました。地震は身近なものであると認識していただき、今後の防災に役立てていただければ幸いです。

## 機能一覧
アクセス時の初期画面表示は過去30日間の地震データが地図上に表示されるとともに、詳細パネルに直近20件の地震情報が表示されます。

### 1. 検索
本Webサイトではこれまで収集した地震情報について「**期間**」、「**地域（震央地名）**」、「**マグニチュード**」、「**震度**」の条件により検索することができます。  
詳細パネルには検索結果のうち直近20件の地震情報が表示されます。

### 2. 地図表示
観測された地震の震央が地図上にピンで表示されます。  
表示されたピンマークを選択することによりピンマークが赤く変化し、その地震の詳細情報が閲覧できます。（2026年04月24日以前は代表地点）

### 3. 検索条件リセット
リセットボタンを押下することで、デフォルト画面の条件（過去30日間の地震データ表示）に戻すことができます。

### 4. 地図拡縮リセット
日本全体ボタンを押下することで、地図表示をデフォルトに戻すことができます。

### 5. データ更新
ページ読み込みから現在までに地震データの更新が発生した場合、データ更新ボタンを押下することでページを再読み込みせずにデータの再取得を行うことができます。
（※内部データの更新は即時ではないため、取得元データへの反映まで地震発生から10～20分程度かかる場合があります。）

### 6. 震度観測点の表示
v1.1.0より震度観測点の表示に対応しました。灰色の点を押下することで観測点の「**名称**」、「**緯度経度**」を確認することができます。

### 7. 震度情報表示（API利用）
直近100件の地震については、詳細パネルから該当地震を選択するとAPIを利用し取得した震度データを取得・表示することができます。また表示された震度マーカーを押下することで「**観測震度**」、「**名称**」、「**緯度経度**」を確認することができます。

## 仕様
- データ収集の都合上、2026年04月24日以前の地震データについて緯度経度は代表地点を使用しているため不正確です。同一震央地名で発生した地震データは重なって表示されますが、拡大縮小した際クラスター表示されたものをクリックすることで個別選択が可能です。
- 地震規模（マグニチュード）が精査により更新された場合、同一日時の地震情報が複数表示される場合があります。（地震システム上の更新の場合）
- 地震規模の更新が発生後しばらく経ってから行われた場合は情報が反映されない場合があります。その際は推定値の情報のみが表示されますが、データの手動更新は原則行いません。（気象庁が会見や資料等で修正した場合）
- データ更新はおおよそ10～20分毎です。データ更新後はページ再読み込みにより変更が反映されます。

## その他
- 使用する地震情報は気象庁から提供されるデータから収集し、まとめたものです。
- 地震情報はGASスクリプトによって私個人のGoogleスプレッドシートに蓄積されます。
- 気象庁によるデータ提供及びGASスクリプトの実行タイミングにより地震情報の更新に時間がかかる場合があります。
- 観測点の緯度経度は[気象庁提供のJSON](https://www.jma.go.jp/jma/kishou/know/jishin/intens-st/stations.json)を利用しています。ただし観測点の情報は静的に保持しているため、情報が更新された場合自動的に更新されず古い情報が表示されます。
- 即時の情報確認には向いていません。過去地震と合わせて確認する際にお役立てください。即時確認には地震速報等、他の手段を検討ください。
